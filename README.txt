This program was more of an initial challenge to put together.
Figuring out how to properly malloc and traverse arrays in the low level environment was more of a
challenge than I expected.

To operate this program, input:
./life rows[int] columns[int] generations[int] filename [print['y'or'n']] [pause['y'or'n']]

To run a test case, try
./life 20 20 100 diehard.txt y y
